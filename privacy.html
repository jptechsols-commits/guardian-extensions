Privacy Policy

Last Updated: December 15, 2025

Overview

Guardian is a parental control browser extension designed to help families monitor and manage browsing activity. This privacy policy explains what data the extension collects and how it is used.

Data Collection

What We Collect

The Guardian extension collects the following data:

  • Browsing Activity: URLs of websites visited in the browser where the extension is installed
  • Device Identifier: A randomly generated unique identifier to associate activity with a specific browser installation
  • Timestamps: When websites are visited
  • User Agent: Browser type and version information

What We Do NOT Collect

  • Passwords or login credentials
  • Form data or personal information entered on websites
  • Cookies or session data
  • Email content or private messages
  • Financial information

Data Storage

All collected data is stored locally on the device or transmitted only to a local agent running on the same computer or local network (127.0.0.1 or LAN address).

No data is transmitted to external servers or third parties.

Data Usage

The collected browsing data is used exclusively for:

  • Enforcing parental control policies (blocking restricted websites)
  • Providing browsing activity reports to parents/guardians
  • Managing access request workflows

Data Sharing

We do NOT share any data with third parties. All data remains within your local network under your control.

Policy Enforcement

The extension enforces browsing policies by:

  • Receiving blocking rules from the local Guardian Agent
  • Blocking access to restricted websites using Chrome's declarativeNetRequest API
  • Displaying a custom blocked page when access is denied

User Rights

Since all data is stored locally and controlled by you:

  • You have full access to all collected data
  • You can delete data at any time by uninstalling the extension or clearing the local database
  • You control the blocking policies through the Guardian Agent

Children's Privacy

This extension is specifically designed for parental control purposes. It is intended to be installed and managed by parents or guardians on devices used by minors.

Changes to This Policy
