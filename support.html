Support

Need help with Guardian? You're in the right place.

Getting Started

  1. Install Guardian Agent: Download and run the Guardian Edge Agent installer on your Windows computer.
  2. Install Extension: The extension will be automatically installed via Chrome policies, or install from Chrome Web Store.
  3. Verify Connection: Click the Guardian icon in your browser toolbar. Status should show "Connected".

Frequently Asked Questions

The extension shows "Disconnected" status

This means the Guardian Agent service isn't running. Open Windows Services (services.msc) and start "Guardian Edge Agent Service". The extension connects to http://127.0.0.1:18181.

Blocked pages aren't working

Check that policies have synced by clicking "Sync Policies" in the extension popup. Verify the agent is serving rules at http://127.0.0.1:18181/v1/policy/rules.

How do I add blocking rules?

Blocking rules are managed through the Guardian Agent, not the extension. The extension fetches and enforces rules from your local agent.

Is my browsing data sent to the cloud?

No. All data is stored locally on your computer or local network. Nothing is ever sent to external servers.

How do I uninstall?

Run the uninstaller from Windows Add/Remove Programs, or run Uninstall.exe from the Guardian installation directory.

Does it work on Edge and Brave?

Yes! Guardian supports Chrome, Microsoft Edge, and Brave browsers. The installer configures policies for all three.
